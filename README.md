# Off Da Branch Site

## Objective

Host the consumer-facing Off Da Branch brand website as a public-safe web property.

## Repository Role

This repository is for the outward-facing brand experience, content, and deployable site assets intended for public presentation.

## Boundary Rules

- Do not store private licensing files, trademark support packages, or raw brand source archives here.
- Do not treat this repository as the source of truth for internal brand governance.
- Keep secrets, infrastructure credentials, and internal-only implementation details out of this repository.

## System of Record

- Public brand site content: this repository
- Private brand/IP source material: `OffDaBranch/brand-system`
- Internal operational tooling: `OffDaBranch/branchops-platform`

## Operating Notes

Use this repository only for public-safe consumer web delivery and supporting assets that are intended to be disclosed publicly.
